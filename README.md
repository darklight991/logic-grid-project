# Logic Grid (Sudoku-Style Puzzle Game)
A C++ console-based puzzle game that generates Sudoku-style puzzles, lets users solve them interactively, validates solutions, and provides hints or automatic solving.

## Features
- Generate Sudoku-style puzzles automatically
- Allow user to input numbers interactively
- Validate the users solution
- Optional difficulty levels (easy, medium, hard)

## Building
```bash
mkdir build
cd build
cmake ..
cmake --build .
